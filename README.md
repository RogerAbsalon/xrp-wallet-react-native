# XRP Wallet - React Native

## 📱 Description
Mobile wallet application developed in React Native connected to the XRP Ledger using official libraries.

## ⚙️ Technologies
- React Native
- JavaScript / TypeScript
- XRP Ledger SDK
- Expo

## 🚀 Features
- Connection to XRP Ledger
- Wallet interaction
- Mobile interface
- Blockchain integration

## 🧠 Project Goal
This project was built to learn blockchain integration and mobile app development with XRP Ledger.
